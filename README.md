# 3D Landing Page Project

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/3-d-landing-page.git
   cd 3-d-landing-page
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Development Server

Start the development server with:
```bash
npm run dev
```

Visit http://localhost:3000 in your browser to view the site.

## Building the Project

To create an optimized production build:
```bash
npm run build
```

## Deployment to Vercel

1. Ensure your project is connected to a Git repository
2. Install Vercel CLI if needed:
   ```bash
   npm i -g vercel
   ````
3. Deploy with production build:
   ```bash
   vercel --prod
   ````

> This project is already configured for Vercel deployment through the `vercel.json` configuration file in the root directory.
